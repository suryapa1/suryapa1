### Architecture & Design Philosophy

```
  Problem Description
         │
         ▼
  ┌─────────────┐     ┌──────────────┐     ┌─────────────┐
  │  Research    │────▶│  Architect   │────▶│  Build      │
  │  (Web + Lit) │     │  (API + DB)  │     │  (TDD-first)│
  └─────────────┘     └──────────────┘     └──────┬──────┘
                                                   │
                              ┌────────────────────┼────────────────────┐
                              ▼                    ▼                    ▼
                       ┌────────────┐      ┌─────────────┐     ┌────────────┐
                       │  Test      │      │  Security   │     │  Verify    │
                       │  (≥80% cov)│      │  (OWASP)    │     │  (E2E)     │
                       └────────────┘      └─────────────┘     └─────┬──────┘
                                                                     │
                                                              Self-Heal Loop
```

> I design systems where AI agents collaborate like engineering teams —
> each agent owns a role, communicates through artifacts, and the pipeline
> self-heals when things break.
